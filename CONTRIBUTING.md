# Contributing to this repository

concrete-rules is currently an independent project.

## Project Status
At this stage, the project is primarily maintained independently. We are not actively seeking external pull requests or major feature contributions at this time.

## Feedback & Issues
If you find a bug or have a suggestion that aligns with the "First Principles" approach of this engine, please feel free to open an Issue for discussion.
